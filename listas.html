Listas

Apesar de listas não serem apresentadas de forma explicita em muitos sites, é uma ferramenta muito útil. Há dois tipos de Listas: ordenadas e não ordenadas

Listas ordenadas

As listas ordenadas são listas onde cada item ocupa uma posição que é indicada pela própria lista:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Também é possível definir o tipo da lista com "type" e o estilo (i,I,A,a,1):

    Romanos
  1. Item I
  2. Item II
  3. Item III
  4. Item IV
    Alfabética
  1. Item A
  2. Item B
  3. Item C
  4. Item D

Listas desordenadas

Listas desordenadas são listas onde os itens não apresentam ordem:

  • Item
  • Item
  • Item
  • Item

Também é possível definir o style de uma lista desordenada como square, circle e disc:

  • Item
  • Item
  • Item
  • Item
